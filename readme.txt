Before After Image slider

== Description ==

This plugin is designed to compare two different images.

=== Features ===
- Easily create a custom Before/After slider.
- Allow Shortcode to enable frontend slider
- Many more to come...

== Installation ==
1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Search for "WP Before After Slider".
3. Click install.
4. Click activate.
